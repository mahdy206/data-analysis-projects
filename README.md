# 📊 Data Analysis Projects

A collection of data analysis projects built during my university studies, covering Power BI dashboards, Python-based EDA, customer segmentation, and data mining techniques.

---

## 📁 Projects

### 1. [Cafe Sales Dashboard](./cafe-sales-dashboard/)
> Power BI · Power Query · DAX

Interactive Power BI dashboard analyzing 10,000 cafe transactions with KPIs, product performance, payment methods, and temporal trends. Includes full data cleaning pipeline.

### 2. [Nike Sales Dashboard](./nike-sales-dashboard/)
> Power BI · Power Query · DAX

End-to-end Power BI analysis of Nike sales data across Indian cities, covering data cleaning, star schema modeling, regional insights, and channel performance.

### 3. [E-Commerce Analysis & Customer Segmentation](./ecommerce-analysis/)
> Python · Pandas · scikit-learn · Plotly

Comprehensive data science project on retail transaction data covering EDA, feature engineering, statistical testing, PCA dimensionality reduction, and K-Means customer segmentation.

### 4. [Data Mining Project](./data-mining/)
> Python · Jupyter Notebook

Academic data mining project applying classification, clustering, or association rule mining techniques to a real-world dataset.
